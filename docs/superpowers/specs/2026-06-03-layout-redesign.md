# AlphaAgents 页面布局重构 — 规格文档

## 背景

当前页面是传统的多 Tab 仪表盘布局（4 个独立页面各自占满）。Agent 化改造后，Chat 成为主交互入口，但老页面（策略、案例、报告）仍然独立存在，右侧的 K 线和工作台需要与聊天联动。需要一个统一布局来承载这些内容。

## 目标

将页面改造为 **ChatGPT 风格的三栏布局**：左侧导航+对话列表、中间 Chat 主区域、右侧可拉收的投研面板（K 线+工作台+筛选结果）。

## 非目标

- 不改动任何后端 API 逻辑
- 不删除老页面的功能，只是重新组织它们的入口和展示方式
- 不引入前端框架，保持原生 HTML/CSS/JS
- 不做多用户、多账户

## 架构设计

### 页面布局

```
┌───────┬──────────────────────────┬──────────────┐
│  Sidebar      │       Main (Chat)        │  Right Panel │
│               │                          │  (可拉收)     │
│  [Alpha]  ────│                          │              │
│               │  ┌────────────────────┐  │  [🔍 展开]   │
│  📊 Chat      │  │ agent: ...         │  │  ──────────  │
│  ⚙ 策略      │  │ 你: 看000001       │  │  K线图       │
│  📚 案例      │  │ agent: [切换K线]   │  │  000001.SH   │
│  📄 报告       │  │                    │  │              │
│  ──────────  │  └────────────────────┘  │  ──────────  │
│               │                          │  个股工作台   │
│  💬 对话1     │  ┌────────────────────┐  │  提醒 操作    │
│  💬 对话2     │  │ [输入框]      [→]  │  │              │
│  💬 对话3     │  └────────────────────┘  │  ──────────  │
│  ──────────  │                          │  筛选结果     │
│  [+ 新对话]   │                          │  共0只        │
└───────┴──────────────────────────┴──────────────┘
```

### 右侧面板结构

```
┌──────────────┐
│ [🔍 展开图表] │  ← 点击后 K 线占据主区域
│ ──────────── │
│              │
│   K 线图      │  ← 固定高度 min-300px，可变
│              │
│ ──────────── │
│  个股工作台   │  ← tab 切换：提醒 | 操作 | 复盘 | 沉淀
│              │
│ ──────────── │
│  筛选结果     │  ← 选股候选列表，可折叠
└──────────────┘
```

### 展开图表模式

点击 `[🔍 展开]` 后：

```
┌───────┬──────────────────────────────┐
│ Side  │         K 线图（全屏）         │
│ bar   │                              │
│       │                              │
│       │                              │
│       │                              │
│  ──── │  ─────────────────────────  │
│       │  [📉 收回]                    │
└───────┴──────────────────────────────┘
```

K 线占据主区域，Chat 和右侧面板隐藏。点击 `[📉 收回]` 恢复三栏。

### 右侧面板的显隐

- 默认**收起**（Chat 占满主区域）
- 右边缘有一个**拉手**（窄竖条），点击或拖拽拉出
- agent 调用股票相关 tool 时**自动滑出**
- 点击面板外区域或 `✕` 按钮收起

### 各个区域的折叠

每个区域（K 线 / 工作台 / 筛选结果）都有一个折叠按钮：
```
┌──────────────┐
│ K 线 ▼  [🔍] │  ← ▼ 点击折叠，▶ 展开
│ ──────────── │
│ 个股工作台 ▼ │
│ ──────────── │
│ 筛选结果 ▶   │  ← 折叠后只显示标题栏
└──────────────┘
```

## 左侧导航栏设计

```
┌──────┐
│Alpha │  ← 品牌
│ ──── │
│ 💬   │  ← Chat（主要入口，始终激活）
│ ⚙    │  ← 策略配置（点击切到策略页，或弹出面板）
│ 📚   │  ← 案例库（点击切到案例页）
│ 📄   │  ← 报告（点击切到报告页）
│ ──── │  ← 分割线
│ 💬1  │  ← 对话历史1
│ 💬2  │  ← 对话历史2
│ 💬3  │  ← ...
│      │
│ [+]  │  ← 新建对话
└──────┘
```

- 对话历史项只显示标题（取对话第一条用户消息或 summary 的前 15 字）
- hover 显示完整标题和日期
- 当前激活的对话高亮

## 老页面整合

| 老页面 | 新位置 |
|--------|--------|
| 看盘选股 | 不再独立存在，由 Chat + 右侧面板替代 |
| 选股策略 | 左侧 nav 按钮，点击在主区域显示策略编辑面板（或弹出侧边栏） |
| 案例库 | 左侧 nav 按钮，点击在主区域显示案例库页面 |
| 分析报告 | 左侧 nav 按钮，点击在主区域显示报告页面 |

当用户点击左侧 nav 的非 Chat 按钮时，主区域切换内容（Chat 隐藏，显示对应页面），右侧面板保持可用的 K 线状态。

## 交互细节

### 拖拽调整面板宽度
- 右侧面板和主区域之间有一条分割线，可拖拽调整比例
- 最小宽度：右侧面板 280px，主区域 400px

### 搜索框
- 放在顶部栏（和"同步数据""执行选股"同一行），始终可见
- 输入即搜，结果下拉

### 响应式
- 窄屏（≤760px）：侧边栏收为顶部横向导航条，右侧面板变为底部弹出面板

## 技术决策与权衡

| 决策 | 选择 | 原因 |
|------|------|------|
| 布局方案 | CSS Grid 三栏 | 灵活，支持拖拽调整 |
| 右侧面板动画 | CSS transition width/transform | 简单，性能好 |
| 图表展开 | 显示/隐藏 Chat + 右侧面板 | 不改变 DOM 结构，只是切换可见性 |
| 拖拽分割线 | 原生 mousedown/move | 不引入第三方库 |
| 老页面迁移 | 渐进式：先保留独立 Tab，再逐步收进 Chat | 避免一次性大重构 |
