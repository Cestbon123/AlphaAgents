# 项目上下文

## 项目定位
AlphaAgents 是一个个人投研助手，用于盘后复盘、选股分析和决策辅助，不涉及交易执行。

## 核心模型
系统按四层组织：
- 策略层：规则选股，生成候选池
- Skill 层：从专家视角进行二次判断
- 提醒层：生成风险与纪律提示
- 画像层：沉淀股票标签与长期笔记

## 架构方向
- Python + FastAPI
- 工作流驱动编排
- 基于 Skill 的能力扩展
- 知识库按原始数据、Wiki 与结构化数据拆分

## 建议目录
- `agents/`
- `workflow/`
- `skills/`
- `knowledge/`
- `api/`

## Agent 风格
优先使用小而专的 agent，例如：
- 策略选股
- 市场复盘
- 个股分析
- 专家评估
- 提醒生成

## 前端说明
- 数据优先，保持高信息密度
- 注重可读性
- 标签与风险信息必须清晰、易扫描
